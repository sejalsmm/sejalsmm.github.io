Coursework

My classes have been filled with challenges, achievement and passion thorugh lab work, programming assignments, and collaborative projects. Check them out below.

18-213 | Computer Systems

This class consisted of multiple labs and assignments to help provide an understanding of how computer systems allow for communication and execution. We learned machine-level code and its generation by optimizing compilers, performance evaluation and optimization, computer arithmetic, building a dynamic memory allocator, networking technology and protocols, supporting concurrent computation, writing a Linux shell program that supports job control and I/O redirection, writing a C program that simulates the behavior of a hardware cache and optimizing a matrix transpose function, write a proxy server supoorts that basic HTTP operation and adding caching to this.

18-290 | Signals & Systems

This class consisted of processing continuous-time and discrete-time signals in both time and frequency domain through tools such as linear time-invariant systems, impulse response, frequency response, convolution, filtering, sampling, Fourier transform, and Laplace transform.

18-220 | Electronic Devices & Analog Circuits

This class consisted of an introduction to semiconductor devices and technology, DC circuit analysis techniques, operational amplifiers, energy storage elements, sinusoidal steady-state response, frequency domain analysis, filters, and transient response of first- and second-order systems. The laboratory component included building a TV jammer, a cell phone charger, a voltage regulator, an RFID scanner, a wireless text message receiver, and using an infrared LED to transmit music over light.

Coursework to look forward to

Next year, I am going more in depth in ECE. Some of the classes I am taking are:

18-330 | Introduction to Computer Security

In this class, I will be learning about the fundamentals of computer security as it relates to Software Security, Cryptography, System/Network Security, and Human Factors.

18-372 | Fundamental of Electrical Power Systems

In this class, I will learn the the general structure of an electric power system, how electric power is addressed from a mathematical point of view, and how to maintain the balance between generation and consumption to avoid catastrophic blackouts.

18-240 | Structure and Design of Digital Systems

In this class, I will learn about real modern digital systems through topics such as boolean algebra, digital number systems and computer arithmetic, combinational logic design and simplification, sequential logic design and optimization, register-transfer design of digital systems, basic processor organization and instruction set issues, assembly language programming and debugging, and a hardware description language.
